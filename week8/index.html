Chapter 8: CSS3 Transforms, Transitions and Animation

  • The CSS3 transform property translate, rotate, scale, and/or skew any element on the page
  • Translation functions move elements left, right, up, or down.
  • Scaling and translation have no impact on the document flow
  • The rotate() function rotates an element around the point of origin by a specified angle value
  • The skew(x,y) function specifies a skew along the x and y axes
  • The transition-property defines the CSS properties of the element that should be transitioned, with all for all properties being the default
  • The transition-duration property sets how long the transition will take: the duration of time it takes to go from the default state to the transitioned state
  • The transition-timing-function controls the pace of the transition
  • We describe the timing function more precisely by defining the cubic-bezier function
  • Negative time delay that's less than the duration of the entire transition will cause it to start immediately
  • A keyframe is a snapshot that defines a starting or end point of any smooth transition.
  • Animation-name property is used to attach an animation to an element
  • The animation-duration property defines the length of time an animation takes to complete one iteration
  • Animation-timing-function determines how the animation will progress over its duration
  • Animation-iteration-count property defines how many times the animation will play through
  • Animation-delay property defines how many milliseconds or seconds to wait before the browser begins the animation
  • Animation-fill-mode property defines what happens before the first animation iteration begins and after the last animation iteration concludes
  • Animation-play-state property defines whether the animation is running or paused

Chapter 12: Canvas, SVG, and Drag and Drop

  • With HTML5's Canvas API, we can draw anything we can imagine, all through JavaScript.
  • The canvas doesn't create a DOM node, is not a text-based format
  • SVG stands for Scalable Vector Graphics, a specific file format that allows to describe vector graphics using XML
  • Images created with SVG are available via the DOM
  • Steps to adding drag and drop
    • Set the draggable attribute on any HTML elements you'd like to be draggable.
    • Add an event listener for the dragstart event on any draggable HTML elements.
    • Add an event listener for the dragover and drop events on any elements that you want to have accept dropped items.
