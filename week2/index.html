WEEK 2 NOTES

Chapter 2: Programming Basics

  • The syntax used by JavaScript is known as a C-style syntax
  • A block is a series of statements that are collected together inside curly braces. They do not need to be terminated by a semicolon.
  • Some reserved words and cannot be used as name variables
  • JS data types: string, symbol, number, boolean, undefined, null, object
  • Special operator called typeof to determine the value type
  • Variables are used in programming languages to refer to a value stored in memory. They have to be declared before they can be used
  • const is used when the variable will not be reassigned to another value
  • let is used if the variable might be reassigned later in the program
  • To assign a value to a constant or variable, we use the = operator
  • Constant and variable names can start with any upper or lower-case letter, an underscore, _, or dollar character, $. They can also contain numbers, but cannot start with them. They are case sensitive.
  • A string is a collection of characters, such as letters and symbols
  • We can access the properties of a string using dot notation and square brackets
  • Template literals use the backtick character, `, to deliminate the string
  • Symbols are the only primitives that don't have a literal form. The only way to create them is to use the Symbol() function
  • Values tend to be set to undefined by JavaScript, whereas values are usually set to null manually by the programmer
  • Bitwise operators work with operands that are 32-bit integers

Chapter 3: Arrays, Logic,a nd Loops

  • Destructuring an array is the concept of taking values out of an array and presenting them as individual values
  • The length property can be used as part of the index to find the last item in an array
  • To remove the last item from an array, use the pop() method
  • The shift() method removes the first item in the array
  • The push() method appends a new value to the end of the array
  • The unshift() method appends a new item to the beginning of the array
  • The concat() method - merge an array with one or more arrays
  • The slice() method creates a subarray
  • The splice() method removes items from an array then inserts new items in their place
  • Reverse the order of an array using the reverse() method
  • Sort the order of an array using the sort() method
  • includes() method returns a boolean value depending on whether the array contains a particular element or not
  • An empty set is created using the new operator and Set() constructor
  • Values can be placed into a set using the add method
  • The number of values in a set can be found using the size() method
  • has() method can be used to check if a value is in a set (boolean)
  • The clear() method can be used to remove all values from a set
  • use the Array.from() method to convert a set into an array
  • To look up a value of map, we can use the get() method
  • The switch statement can be used instead of multiple if and else statements
  • if and else statement are used to control the flow of code
  • A for-of loop can be used to iterate over an array

Chapter 4: Functions

  • Every Function Has a Name
  • Keep Your Code DRY (Dont Repeat Yourself)
  • default parameters are values that will be used by the function if no arguments are provided when it is invoked
  • Default parameters should always come after non-default parameters
  • A function that is passed as an argument to another is known as a callback
