Object Methods: this

  • To access the object, a method can use the this keyword
  • It can be used in any function, even if it's not a method of an object
  • The value of this is defined at run-time

Chapter 5: Objects

  • If a property's value is a function, it is known as a method.
  • An object literal is an object that is created directly in the language by wrapping all its properties and methods in curly braces {}
  • Each property in the object is a key-value pair, separated by commas
  • const spiderman = new Object(); (using constructor function)
  • const spiderman = {}; (empty object)
  • Access the property using dot notation or bracket notation
  • The new Symbol date type can also be used as a computed property key
  • To call an object's method we can also use dot or bracket notation
  • The Object.keys() method will return an array of all the keys of any object
  • Object.values() returns an array of all the object's value
  • Object.entries() returns an array of key-value pair
  • Change the value of an objec's properties using assignment
  • Objects are assigned by reference
  • The parse() method takes a string of data in JSON format and returns a JavaScript object
  • The stringify() method does the opposite, taking a JavaScript object and returning a string of JSON data

Chapter 6: Document Object Model

  • Use the Array.from() method or spread operator to turn a node list into an array
  • Once the node list has been turned into an array, we can use all the array methods on it.
  • getElementsByTagName() will return a live node list of all the elements with the tag name
  • getElementsByClassName() will return a live node list of all elements that have the class name
  • textContent returns the text content of an element as a string
  • Find the text contained inside the text node using the nodeValue method
  • getAttribute() method returns the value of the attribute
  • setAttribute changes the elementâ€™s attributes (two arguments)
  • The classList property is a list of all the classes an element
  • The add method is used to add a class to an element without overwriting any classes that already exist
  • createElement() method that takes a tag name as a parameter and returns that element
  • A text node can be created using the document.createTextNode() method
  • Every node object has an appendChild() method that will add another node
  • An element can be removed from a page using the removeChild() method
  • replaceChild() method can be used to replace one node with another.

Chapter 7 Events

  • The click event occurs when a user clicks with the mouse, presses the Enter key, or taps the screen
  • inline event handlers are best avoided
  • The addEventListener() method is called on a node object, the node to which the event listener is being applied
  • dblclick event occurs when the user doubleclicks on the element to which the event listener is attached
  • touchstart event occurs when a user initially touches the surface
  • touchend event occurs when a user stops touching the surface
  • An event listener can be removed using the removeEventListener() method
  • Bubbling is when the event fires on the element clicked on first, then bubbles up the document tree, firing an event on each parent element until it reaches the root node
  • Capturing starts by firing an event on the root element, then propagates downwards, firing an event on each child element until it reaches the target element that was clicked on
