NOTES


Mobile First Experience

  • Mobile traffic up 18.6% over 2013
  • Responsive retrofitting (cheaper and faster)
  • Mobile First (time consuming and costly, provide best experience across many devices)

How to Improve UX by Understanding Users

  • Be there
  • Be useful
  • Be quick (not only fast loading, functionality is also needed)
  • Reachability of the screen using the thumb for important keys

How to reduce loading speed?

  • Reduce image usage
  • SVG/Webfonts are lower bandwidth
  • Optimize and minify CSS and JS files
  • GZIP files to eliminate unnecessary data

Strategic Menu Design

  • Putting primary control within reach

Mobile-friendly Forms

  • Reduce cognitive load
  • Using float labels
  • Enabling auto fill of the form
  • Enable show or hide password in login form
  • Using the credit card field, maximize the box by including all the necessary data and adding spacing with numbers
  • For the number field, use only number pad
  • Linking phone numbers (tel:)

Mobile Advertising

  • Make it simple and use less space (not 40% plus of the screen)
  • Stop using pop up and auto-playing of videos
  • Enabling auto fill of the form
  • Ads should not be obtrusive
  • Content first
