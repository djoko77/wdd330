WEEK 9 NOTES


Chapter 9: The Window Object

  • The Browser Object Model (or BOM for short) is a collection of properties and methods that contain information about the browser and computer screen
  • Methods of the window object (dialogs)
    • window.alert() method pauses the execution of the program and displays a message in a dialog box
    • window.confirm() method stops execution of the program and displays a confirmation dialog with options of OK or Cancel
    • window.prompt() method stops the execution of the program and displays an input field that allows the user to enter text
  • The Navigator object contains information about the browser being used
    • userAgent returns information about the browser and operating system being used
    • window.location property is an object that contains information about the URL of the current page:
      • reload() method can be used to force a reload of the current page
      • assign() method can be used to load another resource from a URL provided as a parameter
      • replace() method is almost the same as the assign() method, except the current page will not be stored in the session history
      • toString() method returns a string containing the whole URL
  • window.open()method opens a new window
  • window.resizeTo()method to resize a window
  • window.screen object contains information about the screen the browser is displayed on
  • write() method writes a string of text to the page
  • Cookies are small files that are saved locally on a user's computer
  • Applications that contain sensitive information shouldn't rely on cookies expiring using these methods
  • Adding the string ; secure to the end of a cookie will ensure it's only transmitted over a secure HTTPS network

Chapter 14: HTML5 APIs

  • The data- attribute is a way of embedding data in a web page using custom attributes that are ignored by the browser
  • Web Storage API provides a key-value store on the client's computer
    • localStorage is a native object with a number of properties and methods used to store data
    • sessionStorage object is only saved for the current session
    • clear() removes everything stored in local storage
    • removeItem method removes an entry from local storage
    • setItem saves an item locally
  • The Geolocation API is used to obtain the geographical position of the device
    • getCurrentPosition() returns a position object to a specified callback function
    • watchPosition() method calls a callback function every time the position of the device is updated
    • clearWatch() method stops the callback being called
  • Web workers use the concept of messages to communicate back and forth between the main script and worker script
    • postMessage() method sends a message and start the worker working
    • terminate() method stops a completed worker's task within main script or close() method from inside the worker script
    • clearWatch() method stops the callback being called
  • The Service Worker API allows a worker script to run in the background with the added benefit of being able to intercept network requests
  • Websocket is a new protocol that allows two-way communication with a server - also known as push messaging
  • The Notification API allows to show messages using the system's notifications
  • HTML5 introduced the audio and video tags used to insert audio and video clips into a web page
    • An audio clip can be inserted into a page with the <audio> tag, using the src attribute to point to the audio file
    • A video clip can be inserted with the <video> tag, using the src attribute to point to the movie file
    • play() method starts the clip playing from its current position
    • pause() method pauses the clip at its current position
    • volume property is a number to set the audio volume
    • muted property is a boolean value to mute the audio
    • currentTime property is a number value to jump to another part of the clip
    • loop property is a boolean value to true to make the clip repeat in a loop
    • duration property can be used to see how long the clip lasts
