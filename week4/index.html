WEEK 4 NOTES


Chapter 8: Forms

  • The legacy DOM had a useful property called document.forms that returns an HTML collection of all the forms
  • A form object also has a method called elements that returns an HTML collection of all the elements contained in the form
  • We can also access the form controls using their "name" attributes
  • The form.reset() method will reset all the form controls back to their initial values
  • A button with a type attribute of reset can also be used to reset the form
  • The form.action property can be used to set the action attribute of a form
  • Stop the form from being submitted to the URL altogether by using the preventDefault() method
  • In select, the "selected" attribute can be used to set the initial value in the HTML
  • Form validation is the process of checking whether a user has entered the information into a form correctly
  • If the submit button is disabled (disabled property) then no action is taken when it's clicked

Chapter 12: Object-Oriented Programming in JavaScript

  • The parentheses are not required when instantiating a new object using a constructor function
  • The static keyword can be used in class declarations to create a static method
  • Static methods are not available to instances of the class
  • All classes and constructor functions have a prototype property that returns an object
  • Never use arrays or objects as a default value in prototype
  • All objects ultimately inherit from the prototype of the Object() constructor function
  • A class can inherit from another class using the extends keyword in a class declaration
  • The keyword "super" refers to the parent class, and can be used to access any properties and call any methods of the parent class
  • Polymorphism means that different objects can have the same method, but implement it in different ways
  • Instead of using assignment, we can add properties to an object using the Object.defineProperty()
  • A mixin is a way of adding properties and methods of some objects to another object without using inheritance
  • Array.from() method can be used to turn an array-like object into an array
  • Keep inheritance chains short in creating classs

Chapter 15: Modern JavaScript Development

  • A JavaScript library is a piece of code that provides several methods that make it easier to achieve common tasks
  • A big advantage of using jQuery is its support for older browsers like Internet Explorer.
  • Lodash can be thought of as a superset of Underscore as it does everything Underscore does
  • A module is a self-contained piece of code that provides functions and methods that can then be used in other files and by other module
  • A module has its own global scope, so any variables created in the top-level of a module can only be accessed within that module.
  • MVC separates an application into three distinct, independent components that interact with each other: model, view, and controllers
  • React and Vue.js both use the concept of a virtual DOM to speed up the process of updating HTML pages
  • It is usually best to just install the latest version of a package
  • Minification is the process of removing any unnecessary characters from your code to reduce the file size
